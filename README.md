## R-Shiny-Website- Bridging the Gender Gap: A Pathway to Lower Maternal Mortality 
I created an R Shiny website through the code in this repository to summarize my research findings. 
The research project was conducted in Instituto Nacional de Perinatologia, Mexico City, Mexico. The Kiphart Center for Global Health and the Institute of Politics funded this study.
